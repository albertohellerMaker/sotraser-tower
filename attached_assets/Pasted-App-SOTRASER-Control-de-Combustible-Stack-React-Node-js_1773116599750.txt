BLOQUE 8 — IA: agregar TARIFAS como tab interno
═══════════════════════════════════════════════════

Dentro de la pestaña IA agregar tab [TARIFAS]
con estas dos sub-secciones:

[PUNTOS CONFIRMADOS] [TARIFAS POR RUTA]

PUNTOS CONFIRMADOS:
Muestra puntos de puntos_ruta donde confirmado=true
Tabla: TIPO · NOMBRE · COORDENADAS · ACCIONES
Botón "✏ Editar nombre"
Filtro: TODOS / CARGA / ENTREGA / COMBUSTIBLE

TARIFAS POR RUTA:
Tabla de tarifas activas:
RUTA · ORIGEN · DESTINO · KM · LITROS PROM · 
TARIFA CLP · TARIFA USD · ACCIONES

Formulario crear tarifa:
- Nombre ruta (texto libre)
- Origen: dropdown puntos confirmados tipo CARGA
- Destino: dropdown puntos confirmados tipo ENTREGA
- Distancia km (editable)
- Litros promedio (editable)
- Tiempo horas (editable)
- Tarifa CLP (campo principal)
- Tarifa USD (opcional)
- Notas (texto libre)

Crear tablas si no existen:
CREATE TABLE IF NOT EXISTS puntos_ruta (
  id SERIAL PRIMARY KEY,
  camion_id INTEGER REFERENCES camiones(id),
  lat NUMERIC(10,7),
  lng NUMERIC(10,7),
  tipo VARCHAR(20),
  nombre_inferido VARCHAR(200),
  duracion_min INTEGER,
  fecha TIMESTAMP,
  confirmado BOOLEAN DEFAULT false,
  created_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE IF NOT EXISTS tarifas_ruta (
  id SERIAL PRIMARY KEY,
  nombre_ruta VARCHAR(200) NOT NULL,
  punto_origen_id INTEGER REFERENCES puntos_ruta(id),
  punto_destino_id INTEGER REFERENCES puntos_ruta(id),
  origen_nombre VARCHAR(200),
  destino_nombre VARCHAR(200),
  distancia_km NUMERIC(8,1),
  litros_promedio NUMERIC(8,2),
  tiempo_horas NUMERIC(5,2),
  tarifa_clp NUMERIC(12,0),
  tarifa_usd NUMERIC(10,2),
  notas TEXT,
  activa BOOLEAN DEFAULT true,
  created_at TIMESTAMP DEFAULT NOW()
);

Endpoints necesarios:
GET  /api/tarifas/puntos-confirmados
GET  /api/tarifas
POST /api/tarifas
PUT  /api/tarifas/:id
DELETE /api/tarifas/:id
