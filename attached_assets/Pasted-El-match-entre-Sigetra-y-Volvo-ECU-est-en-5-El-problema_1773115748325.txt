LÓGICA NUEVA DE CRUCE — reemplazar la actual:

En lugar de cruzar por ventana de tiempo ±6h,
cruzar por PERÍODO DE VIAJE entre cargas consecutivas:

1. Para cada camión ordenar sus cargas Sigetra 
   por fecha_hora ASC

2. Para cada par de cargas consecutivas (carga_i, carga_i+1)
   del mismo camión definir un viaje:
   - viaje_inicio = fecha_hora de carga_i
   - viaje_fin    = fecha_hora de carga_i+1
   - km_viaje     = km_odometro_i+1 - km_odometro_i
   - litros_sigetra = litros de carga_i+1 
     (lo que cargó para hacer ese viaje)

3. Para ese mismo período buscar en volvo_ecu:
   SELECT SUM(litros_consumidos) as litros_ecu,
          SUM(km_recorridos) as km_ecu
   FROM volvo_ecu 
   WHERE camion_id = X
     AND periodo_inicio >= viaje_inicio
     AND periodo_fin <= viaje_fin

4. Si litros_ecu es NULL (no hay registros ECU 
   en ese período exacto):
   Expandir ventana ±12h y buscar de nuevo.
   Si sigue NULL marcar como "Sin ECU disponible"

5. Calcular diferencia:
   diferencia = litros_sigetra - litros_ecu
   Si diferencia > umbral_critico → CRITICO
   Si diferencia > umbral_alerta  → ALERTA
   Si no                          → OK

6. Calcular rendimiento real del viaje:
   rendimiento = km_viaje / litros_ecu
   (usar litros ECU, no Sigetra, para el rendimiento)

7. Guardar en cuadratura con campos nuevos:
   km_viaje, litros_sigetra, litros_ecu_periodo,
   viaje_inicio, viaje_fin, metodo_cruce
