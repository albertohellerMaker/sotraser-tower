LÓGICA DE CUADRATURA (período = ayer 00:00 → 23:59):

Para cada camión activo con VIN:

  1. VOLVO ECU (fuente verdad):
     litrosQuemadosEcu = 
       MAX(totalFuelUsed WHERE capturedAt <= ayer 23:59)
       - MIN(totalFuelUsed WHERE capturedAt >= ayer 00:00)
     kmEcu = mismo cálculo con totalDistance
     
     Si no hay snapshots de ayer → tieneDatosEcu = false

  2. SIGETRA:
     litrosCargadosSigetra = SUM(litrosSurtidor) 
       WHERE camionId = X AND fecha BETWEEN ayer_inicio AND ayer_fin
     numCargasSigetra = COUNT de esas cargas
     
     Si no hay cargas → tieneDatosSigetra = false, valor = 0

  3. LITROS ESPERADOS:
     Buscar en volvoFuelSnapshots los últimos 90 días para ese VIN
     rendimientoHistorico = totalDistance_90dias / totalFuelUsed_90dias
     litrosEsperados = kmEcu / rendimientoHistorico
     
     Si no hay histórico suficiente → usar 2.8 km/L como fallback
     Corredor = faena.nombre (simplificado, sin clustering por ahora)

  4. CORRECCIÓN ESTANQUE (operación continua):
     nivelInicio = primer snapshot del día (campo fuelLevel si existe, 
                   si no → null)
     nivelFin    = último snapshot del día
     correccionEstanque = nivelFin - nivelInicio (en litros, puede ser 0)
     litrosQuemadosEcuCorregido = litrosQuemadosEcu - correccionEstanque

  5. DELTAS:
     deltaRobo       = litrosCargadosSigetra - litrosQuemadosEcuCorregido
     deltaEficiencia = litrosQuemadosEcuCorregido - litrosEsperados
     rendimientoReal = kmEcu / litrosQuemadosEcuCorregido (si > 0)

  6. CLASIFICACIÓN:
     estadoRobo:
       deltaRobo > 40  → ALERTA
       deltaRobo > 15  → REVISAR
       else            → OK

     estadoEficiencia:
       deltaEficiencia > 50  → ALERTA
       deltaEficiencia > 20  → REVISAR
       else                  → OK

     scoreTotal = 
       (Math.min(Math.abs(deltaRobo) / 40, 1) * 50) +
       (Math.min(Math.abs(deltaEficiencia) / 50, 1) * 30) +
       (estadoRobo === "ALERTA" ? 20 : estadoRobo === "REVISAR" ? 10 : 0)
     scoreTotal = Math.min(Math.round(scoreTotal), 100)
