CAMBIO 1 — Radio adaptativo según distancia
En asignarCorredor() en aprendizaje-engine.ts
═══════════════════════════════════════════════

Reemplazar el radio fijo de 8km por
una función que calcula el radio según
la distancia estimada del corredor:

function getRadioCorredor(
  kmViaje: number | null
): number {
  if (!kmViaje || kmViaje === 0) return 3
  if (kmViaje < 50)   return 1   // urbano/corto
  if (kmViaje < 200)  return 3   // regional
  if (kmViaje < 500)  return 5   // largo
  return 8                        // muy largo
}

En asignarCorredor():

ANTES:
  if (distOrigen < 8 && distDestino < 8)
    → mismo corredor

DESPUÉS:
  const radioKm = getRadioCorredor(viaje.kmGps)
  if (distOrigen < radioKm && distDestino < radioKm)
    → mismo corredor

CAMBIO 2 — Separar por contrato
Los corredores son únicos por contrato
═══════════════════════════════════════════════

Agregar el contrato como parte del match
del corredor.

ANTES:
  Buscar corredor donde origen ±Xkm
  Y destino ±Xkm
  
DESPUÉS:
  Buscar corredor donde origen ±Xkm
  Y destino ±Xkm
  Y contrato = viaje.contrato

  Un corredor Quilicura→CD Vespucio CENCOSUD
  es DISTINTO de
  Quilicura→CD Vespucio ANGLO-CV

  Aunque el origen y destino sean los mismos
  el comportamiento y rendimiento es diferente
  → Deben ser corredores separados

En el schema, si no existe ya:
  ALTER TABLE corredores 
  ADD COLUMN IF NOT EXISTS contrato text

En la query de búsqueda de corredor existente:
  .where(
    and(
      // origen cercano
      // destino cercano
      eq(corredores.contrato, viaje.contrato)
    )
  )

CAMBIO 3 — Nombrar corredores circulares
═══════════════════════════════════════════════

Cuando origen ≈ destino (mismo lugar):
  El corredor NO es "Base → Base"
  Es "Base → [destinos más frecuentes]"

Para corredores donde origenNombre === destinoNombre:

  // Buscar el punto más alejado del GPS
  // en los viajes de ese corredor
  // Ese punto es el destino real del corredor

  const puntoMasLejano = await db.execute(sql`
    SELECT 
      AVG(lat) as lat_medio,
      AVG(lng) as lng_medio,
      MAX(
        6371 * acos(
          cos(radians(${origenLat})) * 
          cos(radians(lat)) * 
          cos(radians(lng) - radians(${origenLng})) + 
          sin(radians(${origenLat})) * 
          sin(radians(lat))
        )
      ) as max_dist
    FROM geo_puntos
    WHERE patente = ANY(${patentes})
      AND timestamp_punto BETWEEN ${fechaInicio} 
      AND ${fechaFin}
  `)
  
  // Geocodear ese punto contra geocercas
  const destinoReal = await geocodearPunto(
    puntoMasLejano.lat_medio,
    puntoMasLejano.lng_medio
  )
  
  // Renombrar el corredor
  nombre = `${origenNombre} → ${destinoReal}`
