CONTEXTO DE LOS DATOS
═══════════════════════════════════════════════════

La app tiene camiones asignados a múltiples
contratos/faenas: CENCOSUD, ANGLO-COCU, CODELCO,
BHP, GASCO y otros.

Los camiones están en la tabla "camiones".
El campo que los asocia al cliente puede ser:
- faena
- contrato  
- cliente
(usar el que exista en la tabla)

Las cargas están en "cargas_tarjeta" con campo
"cliente" que contiene el nombre del contrato.

Los viajes están en "tms_viajes" con campo
"cliente_sigetra" o "contrato_id".

La telemetría está en "volvo_ecu" asociada
a camion_id.

PASO 1 — IDENTIFICAR CAMIONES CENCOSUD
═══════════════════════════════════════════════════

Primero identificar qué camiones pertenecen
a CENCOSUD. Puede ser por:

1. Campo directo en tabla camiones:
   WHERE faena ILIKE '%CENCOSUD%'
   OR cliente ILIKE '%CENCOSUD%'
   OR contrato ILIKE '%CENCOSUD%'

2. Por cargas en Sigetra:
   SELECT DISTINCT patente FROM cargas_tarjeta
   WHERE cliente ILIKE '%CENCOSUD%'

3. Por asignación en faenas:
   JOIN faenas ON camiones.faena_id = faenas.id
   WHERE faenas.nombre ILIKE '%CENCOSUD%'

Usar el método que funcione según la estructura
real de la BD. Si hay múltiples, combinarlos.

Crear una función helper en el backend:
getCencosudCamionIds() → devuelve array de IDs
getCencosudPatentes() → devuelve array de patentes

Estas funciones se usan en TODOS los endpoints.

═══════════════════════════════════════════════════
PASO 7 — NÚMERO REAL DE CAMIONES CENCOSUD
═══════════════════════════════════════════════════

Ejecutar esta query para saber cuántos son:

SELECT COUNT(DISTINCT c.id) as total,
  COUNT(DISTINCT ct.patente) as con_sigetra
FROM camiones c
LEFT JOIN cargas_tarjeta ct 
  ON ct.patente = c.patente
  AND ct.cliente ILIKE '%CENCOSUD%'
WHERE c.faena ILIKE '%CENCOSUD%'
  OR c.cliente ILIKE '%CENCOSUD%'
  OR ct.patente IS NOT NULL;

Si la query devuelve 0 (porque el campo
de asignación se llama distinto), intentar:

SELECT DISTINCT patente, COUNT(*) as cargas
FROM cargas_tarjeta
WHERE cliente ILIKE '%CENCOSUD%'
GROUP BY patente
ORDER BY cargas DESC;

Usar el resultado para confirmar qué patentes
son CENCOSUD y aplicar el filtro correcto.
